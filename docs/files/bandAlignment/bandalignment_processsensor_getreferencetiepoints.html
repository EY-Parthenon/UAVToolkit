BandAlignment_ProcessSensor_GetReferenceTiePoints

Simple procedure that will generate tie points base don the input group.

Syntax

BandAlignment_ProcessSensor_GetReferenceTiePoints, group, parameters, [ RASTER = enviraster ], [ /NO_OUTPUT ], [ FILTERED_TIEPOINTS = bandalignmenttiepoints ]

Arguments

GROUP

in, requried, type=string[*]

Array of strigns that represents the files to process.

PARAMETERS

in, required, type=dictionary

The dictionary parameters used for all the band alignment processing.

Keywords

RASTER

in, optional, type=ENVIRASTER

If set, then we wont stack the images into a raster. Can save a small amount of time.

NO_OUTPUT

in, optional, type=boolean, default=false

If set, then no output files will be created. Otherwise a .dat file and tie points will be saved to disk. Used for secondary image registration.

FILTERED_TIEPOINTS

out, optional, type=bandAlignmentTiePoints

Returns a direct reference to the tie points that are generated by this procedure.
